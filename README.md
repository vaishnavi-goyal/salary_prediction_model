# 💼 AI Salary Prediction Dashboard

An AI-powered Salary Prediction Web Application built using Machine Learning and Streamlit. The application predicts employee salaries based on professional and educational information while providing interactive analytics, AI insights, and downloadable reports.

---

## 🚀 Features

- 💰 AI-based Salary Prediction
- 📊 Interactive Dashboard
- 🌍 Multi-Currency Salary Conversion
- 📈 Salary Analytics with Plotly Charts
- ⭐ Candidate Profile Rating
- 🤖 AI Career Recommendations
- 📄 Download Salary Report (CSV/PDF)
- 🕒 Prediction History
- 🎨 Modern & Responsive UI
- ⚡ Real-Time Prediction

---

## 🧠 Machine Learning Model

The prediction is based on the following features:

- 💼 Job Title
- 🎓 Education Level
- 🏢 Industry
- 🏬 Company Size
- 🌍 Location
- 🏠 Work Mode (Remote / Hybrid / On-site)
- 📈 Years of Experience
- 🛠 Skills Count
- 📜 Certifications
